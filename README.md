# 까꿍 놀이 (Peekaboo Game) 👶

아기를 위한 인터랙티브 까꿍 놀이 웹 게임입니다.

## 🎮 데모

**[지금 바로 플레이하기](https://lazyduo.github.io/chisu-peekaboo/)**

## ✨ 특징

- 🚗 **테마 선택**: 탈 것, 동물, 음식, 전체 중 선택 가능
- 🎵 **음향 효과**: Web Audio API를 사용한 실시간 사운드 생성 (외부 파일 불필요)
- 🎨 **귀여운 캐릭터**: 150개 이상의 다양한 이모지 캐릭터
- ✨ **시각 효과**: 부드러운 애니메이션과 시각적 피드백
- 📱 **반응형 디자인**: 모바일, 태블릿, PC 모두 지원
- 🎯 **점수 시스템**: 클릭할 때마다 점수가 올라가고 10점마다 축하 사운드

## 🎯 게임 방법

1. 원하는 테마 선택 (탈 것, 동물, 음식, 전체)
2. 화면에 나타나는 캐릭터를 클릭!
3. "까꿍!", "야호!" 같은 재미있는 반응 확인
4. 점수를 쌓아가며 즐기기

## 🎨 테마

### 🚗 탈 것
자동차, 기차, 비행기, 배 등 35가지 탈 것 캐릭터

### 🐶 동물
강아지, 고양이, 공룡, 물고기 등 47가지 동물 캐릭터

### 🍎 음식
과일, 간식, 요리 등 67가지 음식 캐릭터

### 🌈 전체
모든 캐릭터 포함 (150개 이상)

## 🔊 음향 효과

- 캐릭터 등장 소리 (올라가는 톤)
- 클릭 효과음
- 메뉴 선택 멜로디
- 10점마다 축하 사운드
- 배경 클릭 반짝임 소리

## 🛠️ 기술 스택

- **HTML5**: 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (ES6+)**: 게임 로직
- **Web Audio API**: 음향 효과 생성
- **GitHub Pages**: 배포

## 📦 로컬에서 실행하기

```bash
# 저장소 클론
git clone https://github.com/lazyduo/chisu-peekaboo.git

# 디렉토리 이동
cd chisu-peekaboo

# 브라우저에서 index.html 파일 열기
open index.html  # macOS
# 또는
start index.html  # Windows
# 또는
xdg-open index.html  # Linux
```

## 🌟 특별한 점

- **외부 의존성 없음**: 단일 HTML 파일로 완성
- **오프라인 작동**: 한 번 로드되면 인터넷 연결 없이도 사용 가능
- **가벼움**: 전체 파일 크기 약 15KB
- **즉시 플레이**: 설치나 로딩 없이 바로 시작
